# Redis
```sh
# Acessa o cli do redis
$ redis-cli

# Mostra todas as keys do redis
$ keys *

# Retorna todas as informações da chave
$ hgetall <chave-retornada-do-key-*>

# Retorna apenas os dados da chave
$ hget <chave-retornada-do-key-*> data

# o hget é para ser usada para chave-valor que são do tipo hash
```
